# Book store management

## I. How to run project

**Front-end:** `localhost:3000`

**Back-end:** `localhost:8080`

**MySQL:**

- Host: `127.0.0.1:39062`

- User: `root`

- Password: `123456`

- DB Name: `bookstoremanagement`

### 1. Move to docker-compose folder

```bash
cd docker-compose
```

### 2. Run docker-compose

```bash
docker-compose up
```
